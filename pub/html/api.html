TREE JS API

To use TreeJS API, include these two files in your html head element preceeding scripts which use TreeJS


    <script type="text/javascript" src="https://treejs309.herokuapp.com/pub/Tree.js">
    <link rel="stylesheet" href="https://treejs309.herokuapp.com/pub/Tree.css">
TreeJS

global object containing top level library methods, property of the window object

Property Type Description
Tree class main object of TreeJS library used to represent a Tree (see Tree)
TreeNode class object representing the nodes of an instantiated Tree (see Node)
setOptionsDefault function used to set properties of the default options object that sets object options that are not specified
addTreeAsChild function Creates a Tree's dom representation if not already created and adds a Tree object's dom representation to the dom

@param tree: The Tree object to append to the DOM
@param parent: the parent / containing element that the Tree's Dom representation will become a child of
Tree

Property Type Description
constructor function instantiate a Tree object
parameters: (rootid, structure, options)

@param rootid: String used as the id for the root container that will be created to hold
@param structure: structure object
@param options: Object holding all options for this Tree (see options object)

root function returns the ith root node of this Tree

@param i: integer greater than 0, used to index the root node's of the tree
collapse function collapse the entire tree so that only the root nodes are showing

sets the display css property of all children's dom elements to "none"
expand function expands the root nodes so that their children are showing

sets the display css property of all children to their previous display styles before they were set to "none" on a collapse call
expandAll function recursivley expands all nodes of the Tree so the entire Tree's nodes are visible
structure Object deep copy of the structure used in constructing this Tree object (see structure)
domNode Element div element containing this Tree's dom representation, id of this div is set to the id passed into the constructor
